# 快速開始指南

## 5 分鐘快速啟動

### 1. 安裝依賴
```bash
pip3 install -r requirements.txt
```

### 2. 生成測試資料
```bash
python3.11 generate_sample_data.py
```

### 3. 啟動應用程式
```bash
streamlit run app.py
```

應用程式將在 `http://localhost:8501` 開啟

## 功能導覽

### 📊 首次使用

1. **選擇資料來源**
   - 側欄選擇「手動上傳」或「Google Drive 同步」
   - 上傳或同步三個必要檔案

2. **查看 KPI**
   - 精品滲透率、美容轉換率、銷售額等
   - 年度目標進度條

3. **分析資料**
   - 選擇時間範圍、集團、經銷商
   - 切換「經銷商/集團分析」或「產品分析」視圖

### 🎯 關鍵功能

#### 時間篩選
- **本月**：當月 1 日至今
- **本季**：當季開始至今
- **本年**：1 月 1 日至今
- **自訂範圍**：選擇開始和結束日期

#### 分析視圖
- **經銷商/集團分析**
  - 銷售趨勢圖表
  - 集團銷售比較
  - 經銷商排名

- **產品分析**
  - 按銷售額排名前 10
  - 按數量排名前 10
  - 產品績效矩陣

#### AI 洞察
- **執行摘要**：自動生成銷售摘要
- **異常偵測**：識別下降 > 20% 的產品
- **策略建議**：精品策略建議

#### 匯出選項
- **Excel 匯出**：下載篩選後的銷售資料
- **完整報告**：生成包含圖表的報告

## 常見問題

### Q: 如何上傳自己的資料？
A: 
1. 確保檔案名稱完全匹配：`Boutique_Raw.xlsx`、`Beauty_Raw.xlsx`、`Target_2026.xlsx`
2. 確保所有必要欄位都存在
3. 在側欄選擇「手動上傳」並上傳檔案

### Q: 如何設定 Google Drive 同步？
A:
1. 建立 Google Cloud 專案並啟用 Drive API
2. 下載 OAuth 2.0 認證 JSON
3. 放在 `.google_credentials.json`
4. 在側欄選擇「Google Drive 同步」

### Q: AI 洞察功能需要什麼？
A:
- 需要設定 OpenAI API Key
- 設定環境變數：`export OPENAI_API_KEY=your_key`
- 或在 `.env` 檔案中設定

### Q: 如何匯出資料？
A:
1. 篩選所需資料
2. 點擊「匯出精品銷售 (Excel)」或「匯出美容銷售 (Excel)」
3. 瀏覽器會自動下載 Excel 檔案

## 效能優化建議

### 大資料集
- 使用時間篩選減少資料量
- 按集團或經銷商篩選
- 避免一次載入超過 1 年的資料

### 圖表優化
- 使用「按銷售額排名」而非「按數量排名」以提高效能
- 限制排行榜為前 10-20 項

## 故障排除

### 應用程式無法啟動
```bash
# 檢查 Python 版本
python3.11 --version

# 重新安裝依賴
pip3 install --upgrade -r requirements.txt

# 清除快取
rm -rf ~/.streamlit
```

### 資料無法載入
- 檢查檔案名稱是否正確
- 確認 Excel 檔案未損壞
- 驗證所有必要欄位都存在

### Google Drive 認證失敗
- 清除舊 token：`rm .google_drive_token.pickle`
- 檢查認證 JSON 檔案位置
- 確認 API 已啟用
